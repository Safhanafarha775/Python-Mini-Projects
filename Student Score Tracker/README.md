# Student Score Tracker

A simple Python mini project (45–50 minutes) that asks the user to input 5 marks, stores them in a list, and calculates:

- Total marks  
- Average marks  
- Highest mark  
- Lowest mark  
- Pass/Fail (Pass if average ≥ 40)

---

## Features
- Takes 5 subject marks as input  
- Uses list to store marks  
- Calculates total and average  
- Finds the highest and lowest marks using built-in functions  
- Determines Pass/Fail based on average  

---

## Technologies Used

Python 3
Lists
Loops
Built-in functions: sum(), max(), min()
Conditional statements

---

## Learning Outcomes

Taking multiple inputs from the user
Storing values in a list
List operations
Basic data analysis using Python
